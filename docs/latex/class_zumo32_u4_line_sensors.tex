\doxysubsection{Detailed Description}
Gets readings from the five down-\/facing line sensors on the front sensor array. 

The functions that this class inherits from \mbox{\hyperlink{class_q_t_r_sensors_r_c}{QTRSensors\+RC}}, such as read() and calibrate(), are documented in the \href{https://www.pololu.com/docs/0J19}{\texttt{ user\textquotesingle{}s guide for the QTRSensors library}}. QTRSensors is a separate Arduino library with a \href{https://github.com/pololu/qtr-sensors-arduino}{\texttt{ separate Git\+Hub repository}}, but we include a copy of it in the Zumo32\+U4 library since it is needed for the \mbox{\hyperlink{class_zumo32_u4_line_sensors}{Zumo32\+U4\+Line\+Sensors}} class.
